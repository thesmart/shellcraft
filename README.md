# Shellcraft Agent Skills

A set of agent skills for writing and testing POSIX-compliant shell scripts (i.e. `sh`, not `bash`,
`zsh`, etc.).

## Supported Distributions

This project aims to support:

AlmaLinux, Alpine Linux, Amazon Linux 2 / Amazon Linux 2023, Arch Linux, Bottlerocket, CBL-Mariner
/ Azure Linux, CentOS Stream, Container-Optimized OS, Debian, Fedora, Flatcar Container Linux,
Gentoo, NixOS, openSUSE Leap, openSUSE Tumbleweed, Oracle Linux, Red Hat Enterprise Linux (RHEL),
Rocky Linux, Slackware, SUSE Linux Enterprise (SLES), Ubuntu LTS, Ubuntu Pro, Void Linux, Wolfi

---

## Installation

### Skillfish

```sh
skillfish add thesmart/shellcraft/skill
```

### Manual

Clone into your Claude Code skills directory:

```sh
git clone https://github.com/thesmart/shellcraft.git ~/.claude/skills/shellcraft
```
